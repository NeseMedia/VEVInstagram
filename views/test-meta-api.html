üìö Required Setup in Meta Dashboard

  1. Create Instagram Basic Display App

    Go to Meta for Developers

    Create new app ‚Üí Select "Consumer" ‚Üí Add Instagram Basic Display

  2. Configure OAuth Redirect URIs

    Add these URIs to your Instagram Basic Display settings:

    https://vevinstagram-production.up.railway.app/auth/instagram/callback https://localhost:3000/auth/instagram/callback (for local testing)

  3. Add Instagram Test Users

    In Roles ‚Üí Instagram Testers ‚Üí Add Instagram Testers

    The test user must accept the invitation from Instagram Settings

  4. Get App Credentials

    From Instagram Basic Display ‚Üí Basic Display:

    • Instagram App ID
    • Instagram App Secret
    • Client Token (generate one)

  5. Set Environment Variables in Railway

    Add these to your Railway project variables:

    INSTAGRAM_APP_ID=your_app_id INSTAGRAM_APP_SECRET=your_app_secret INSTAGRAM_REDIRECT_URI=https://vevinstagram-production.up.railway.app/auth/instagram/callback
